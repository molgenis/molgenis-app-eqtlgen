Changelog
2024-02-23: Initial data release
Raw data
Processed (de-anonymized) scRNA-seq data of the STEMI samples, including a text file that links each cell barcode to its respective individual, has been deposited at the European Genome-Phenome Archive (EGA), which is hosted by the EBI and the CRG, under accession number EGAD00001010064. The 38 control samples come from the 1m-scBloodNL dataset.

The request for the STEMI cohort raw data requires a guest account at the UMCG, which can be done here. After obtaining a UMCG guest account, the data can be requested here, or in the form below. A signed DAC is required, which can be found here
